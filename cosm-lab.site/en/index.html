Research topics
Following topics can be used as a topics for master and bachelor students projects. Also they can be taken as a topic for practical work asessments. Working on any of the topic assumes interaction with related project teams and planned for 10-15 work hours per-week with flexible schedule. Here are several topics examples:

SDC-01
Designing and development of sub-system, allowing execution of system dynamics models on GPU
In terms of this topic it is required to investigate existing approaches for generation models executable code (on C and Python) and prepare a system that will translate definitions of system dynamics models into code for GPU (i.e. for CUDA)
SDL-02
Research of approaches for time-series analysis with application of machine learning. Solving problems of correlation detection, anomlies detection and predicitons.
Frequently, especially when performing monitoring, collected data are hard to analyze and identify relationships between measurements. For such problems, machine learning methods fit pretty well. In terms of this topic it is required to research available ML mechanisms, implement a solution to solve mentioned problems and integrate this solution into sdLitica platfrom. This project is developed together with partners from Thapar University (India) and UNISINOS (Brazil).
IPR-01
Implementation of data extraction from varios tools for project management and tracking
Main goal of project InsightProject is aggregation and analysis of wide range of data extracted from tools used to support process of software development. Among these tools we rely on bug trackers (Mantis, Jira, GitHub issues), version control systems, continuous integration and delivery tools, wiki engines and others. To work with each tool we to develop an adapter to get data properly.
